## 功能介绍（完整见运行截图）
管理员：登录和注册，用户管理，音乐和歌曲管理，订单管理，支付及系统管理。管理页面包括上传和管理音乐视频音频、设置定价、以及处理订单审核。支持歌手和用户信息的查看编辑，提供即时客服支持。用户：注册和登录，访问个性化音乐库、付费音乐、及音乐资讯。通过个人中心可以更新个人信息，管理订单和收藏。用户也可通过多种支付方式完成在线音乐购买。在主界面查看和评论音乐，使用内置播放器控制音乐播放。
